# Bridge Essays

This folder is the upload target for the twelve prose bridge essays connecting **The First Fixed Opening** and **Settlement Gate**.

Upload the essay HTML files directly into this folder.

## Required naming

Use two-digit numeric prefixes so the final collection can be wired in order:

- `01.html`
- `02.html`
- `03.html`
- `04.html`
- `05.html`
- `06.html`
- `07.html`
- `08.html`
- `09.html`
- `10.html`
- `11.html`
- `12.html`

If the files already have descriptive names, keep the numeric prefix, for example:

- `01-fixed-opening-to-settlement.html`
- `02-affordance-and-receipt.html`

## After upload

Once all twelve files are present, the collection should be wired into:

- a bridge index page,
- the root landing page,
- the eBook banner/card navigation,
- the Settlement Gate white paper navigation,
- and the README project map.
